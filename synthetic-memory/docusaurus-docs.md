# Docusaurus Documentation Site

## Setup

```bash
npx create-docusaurus@latest my-docs classic
cd my-docs
npm start
```

The "classic" template includes docs, blog, and custom pages.

## Configuration

```javascript
// docusaurus.config.js
module.exports = {
  title: 'My Docs',
  url: 'https://docs.example.com',
  baseUrl: '/',
  organizationName: 'myorg',
  projectName: 'my-docs',
  themeConfig: {
    navbar: {
      title: 'My Docs',
      items: [
        { type: 'docSidebar', sidebarId: 'docs', label: 'Docs' },
        { to: '/blog', label: 'Blog' },
        { href: 'https://github.com/myorg/repo', label: 'GitHub' },
      ],
    },
    footer: { style: 'dark', links: [/* ... */] },
    prism: { theme: require('prism-react-renderer').themes.github },
  },
  presets: [['classic', {
    docs: { sidebarPath: require.resolve('./sidebars.js'), editUrl: 'https://github.com/myorg/repo/edit/main/' },
    blog: { showReadingTime: true },
    theme: { customCss: require.resolve('./src/css/custom.css') },
  }]],
};
```

## Sidebars

```javascript
// sidebars.js
module.exports = {
  docs: [
    'intro',
    { type: 'category', label: 'Tutorial', items: ['tutorial/step1', 'tutorial/step2'] },
    { type: 'autogenerated', dirName: 'api' },  // auto-generate from folder
  ],
};
```

Use `autogenerated` for large doc sets. Control order with `sidebar_position` in front matter.

## MDX Features

Docusaurus supports MDX (Markdown + JSX):
```mdx
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
  <TabItem value="npm" label="npm">npm install foo</TabItem>
  <TabItem value="yarn" label="yarn">yarn add foo</TabItem>
</Tabs>
```

Built-in components: `Tabs`, `CodeBlock`, `Admonitions` (:::note, :::tip, :::warning, :::danger).

## Versioning

```bash
npm run docusaurus docs:version 1.0
```

Creates a snapshot in `versioned_docs/version-1.0/`. The `docs/` folder becomes "Next" (unreleased).

## Search

- **Algolia DocSearch**: Free for open-source. Configure `algolia` in `themeConfig`.
- **Local search**: Use `@easyops-cn/docusaurus-search-local` plugin.

## Deployment

```bash
npm run build          # generates static site in build/
GIT_USER=myuser npm run deploy  # GitHub Pages
```

Or deploy `build/` to any static host (Vercel, Netlify, Cloudflare Pages).

## i18n

```bash
npm run docusaurus write-translations -- --locale ko
```

Translated docs go in `i18n/ko/docusaurus-plugin-content-docs/current/`.
